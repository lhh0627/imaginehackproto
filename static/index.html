Track 2 : Hilti Cloud

Secure & Energy-Aware Cloud Sentinel

A hackathon prototype that watches fake construction cloud workloads, flags cyber exposure and carbon-heavy idle jobs, then triggers an automated fix through Python into Docker.

Demo flow

  2. Python scans containers, exposed ports, and energy labels.
  3. Dashboard shows the warning and recommendation.
  4. Auto-Fix sends a command back through Python to remove the risky workload.
